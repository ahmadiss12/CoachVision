# Squat Exercise Pose Dataset

This dataset contains **pose-based features extracted from squat exercises**, designed for **machine learning classification** of correct vs. incorrect squat forms. The data is derived from videos of correct squats and augmented synthetically to represent common errors.

---

## Dataset Overview

- **Total examples:** Derived from original video frames and synthetically augmented
- **Features:** Joint angles and positions extracted from Mediapipe Pose landmarks
- **Labels:** Classification of squat form

| Label | Description                   |
|-------|-------------------------------|
| 0     | Correct squat form            |
| 1     | Shallow squat                 |
| 2     | Forward lean                  |
| 3     | Knees caving in               |
| 4     | Heels off ground              |
| 5     | Asymmetric squat              |

---

## Features (Columns)

- `left_knee_angle`, `right_knee_angle` — Knee joint angles  
- `left_hip_angle`, `right_hip_angle` — Hip joint angles  
- `left_ankle_angle`, `right_ankle_angle` — Ankle joint angles  
- `spine_angle` — Angle of the spine relative to the hip  
- `torso_lean` — Forward/backward lean of the torso  
- `left_knee_lateral`, `right_knee_lateral` — Lateral deviation of the knees  
- `symmetry_score` — Sum of differences between left and right angles  
- `hip_depth` — Minimum vertical position of the hips  
- `video_file` — Original video filename (for reference)  
- `frame` — Frame number in the original video  
- `label` — Class label (see table above)

---

## Data Generation

1. **Correct squats:** Pose keypoints extracted from YouTube videos using [Mediapipe Pose](https://google.github.io/mediapipe/solutions/pose.html).  
2. **Incorrect squats:** Synthetic errors generated by perturbing key angles and positions numerically.  

This approach allows training ML models without sharing copyrighted video content.
